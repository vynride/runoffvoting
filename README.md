## Usage:
To host your own Runoff election:  
`./runoff Candidate_1 Candidate_2 Candidate_3`

## Example:    
```
./runoff Alice Bob Charlie

Number of voters: 4
Rank 1: Alice
Rank 2: Bob 
Rank 3: Charlie

Rank 1: Alice
Rank 2: Charlie
Rank 3: Bob

Rank 1: Charlie 
Rank 2: Alice 
Rank 3: Bob

Rank 1: Bob
Rank 2: Alice
Rank 3: Charlie

Alice
```
